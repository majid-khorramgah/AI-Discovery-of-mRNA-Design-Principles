# AI Discovery of mRNA Design Principles

## Overview

Current AI methods optimize mRNA sequences, but the biological principles governing optimal mRNA design remain poorly understood.

This project aims to develop an explainable AI framework that learns hidden design rules from natural protein-mRNA pairs and enables generalizable mRNA optimization.

---

## Research Question

Can AI discover biological principles underlying natural mRNA design?

---

## Hypothesis

Natural mRNA sequences contain hidden evolutionary optimization patterns that can be learned using representation learning and explainable AI.

---

## Dataset

19,037 human protein-mRNA pairs

Sources:
- UniProt
- NCBI RefSeq

Features:
- Protein sequence
- mRNA sequence
- GC content
- Codon usage
- CAI
- RNA secondary structure
- MFE

---

## Methodology

Protein representation learning
↓

↓
Explainable AI
↓

Multi-objective optimization
